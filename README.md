## 🎯 Core Features
- 📱 **Responsive Design** – Mobile‑first layout, works on all devices
- 🧭 **Interactive Navigation Menu** – Scroll & hover effects with glassmorphism

- 🎨 **Modern UI** – Futuristic gradients, animations, & particle backgrounds
